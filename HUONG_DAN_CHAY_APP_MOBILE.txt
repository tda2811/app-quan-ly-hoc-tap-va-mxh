# HƯỚNG DẪN CHẠY VÀ XUẤT FILE ỨNG DỤNG MOBILE (REACT NATIVE)

Chào bạn, xin khẳng định lại **đây 100% là ứng dụng Mobile (Native App)** dành cho Android và iOS. React Native (với framework Expo) là công nghệ mà Facebook, Instagram, Shopee đang sử dụng để viết App điện thoại.

Lý do khi nén nút "Đăng Nhập" bạn gặp lỗi **"Lỗi kết nối máy chủ!"** là vì:
1. Bạn chưa bật Server Backend API (Node.js).
2. Địa chỉ IP trong App đang là IP giả định `192.168.1.100`, nó chưa trỏ đúng vào IP máy tính của bạn.

Dưới đây là các bước chuẩn xác nhất để chạy App này trên điện thoại thật của bạn.

---

## BƯỚC 1: KHỞI ĐỘNG MÁY CHỦ API (BACKEND) VÀ DATABASE
Ứng dụng trên điện thoại không thể tự kết nối trực tiếp với MySQL (để bảo mật), nó bắt buộc phải gọi lên một máy chủ API trung gian (Backend).

1. Bạn mở ứng dụng XAMPP (hoặc MySQL Workbench) và bật **MySQL** lên. Đảm bảo bạn đã Import file `database_schema.sql` vào tạo database có tên `student_lms_social`.
2. Mở một Terminal mới trong VS Code, đi vào thư mục Backend và chạy nó:
   ```cmd
   cd backend
   npm install
   npm run dev
   ```
   *Thấy thông báo: `🚀 Server Backend đã chạy tại http://localhost:3000` là thành công.*

---

## BƯỚC 2: CẬP NHẬT ĐỊA CHỈ IP LAN CHO APP MOBILE
Khi App cài vào điện thoại của bạn, nó không hiểu `localhost` là gì, nó cần biết địa chỉ IP thật của máy tính (đang phát chung Wifi).

1. Mở Terminal mới thứ 2, gõ lệnh `ipconfig` (trên Windows) dư để lấy IP. Tìm dòng **IPv4 Address** (Ví dụ kết quả là: `192.168.1.15`).
2. Mở file `EduConnectApp/src/services/authService.js`.
3. Sửa lại chuỗi URL cho đúng với IP bạn vừa tìm thấy:
   ```javascript
   // Đổi dòng này
   const API_URL = 'http://192.168.1.100:3000/api';
   // Thành (Thay số thành IP của máy tính bạn)
   const API_URL = 'http://192.168.1.15:3000/api'; 
   ```

---

## BƯỚC 3: CHẠY APP LÊN ĐIỆN THOẠI THẬT ĐỂ TEST (Cách lập trình viên hay dùng)
Đây là cách test mượt mà và thực tế nhất:

1. Rút điện thoại thật của bạn ra, truy cập App Store (iOS) hoặc CH Play (Android).
2. Tìm và tải ứng dụng có tên: **Expo Go** (Miễn phí). Đảm bảo điện thoại bắt cùng 1 mạng Wifi với máy tính.
3. Mở một Terminal mới thứ 3 trên VS Code, đi vào thư mục App và khởi động bộ biên dịch Mobile:
   ```cmd
   cd EduConnectApp
   npx expo start
   ```
4. Trên màn hình máy tính của bạn sẽ hiện ra một **Mã QR Code lớn**.
   * Mở điện thoại, vào App **Expo Go**.
   * Bấm nút **Scan QR Code**.
   * Quét QR trên màn hình máy tính. Đợi 10 giây để code nạp vào máy.
👉 **Tada! Ứng dụng EduConnect hoàn toàn gốc đã nằm trong điện thoại của bạn.** Bạn sờ chạm, vuốt trên điện thoại cực mượt. Ấn thao tác Đăng nhập lúc này App sẽ tự động phi từ điện thoại vào IP máy tính để xử lý thành công!
